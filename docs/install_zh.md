# 如何安装

在该页面[releases](https://github.com/wanglin2/obsidian-simplemindmap/releases)下载最新版本的插件文件，只需下载：`main.js`、`manifest.json`、`styles.css`这三个文件。

下载完成后打开 Obsidian 仓库根路径下的 `/.obsidian/plugins/` 文件夹，如果已经安装过第三方插件，可以直接进 Obsidian 软件，点击设置 -> 第三方插件 -> 已安装插件，右侧有个文件夹图标，点击即可打开该文件夹。

打开该文件夹后，新建一个文件夹，名称为`obsidian-simplemindmap`，然后将下载的三个文件复制到该文件夹中。

再次打开 Obsidian 软件，点击设置 -> 第三方插件 -> 已安装插件，点击右侧的刷新图标按钮，即可看到该插件，点击启用即可。
